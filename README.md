### 🌐 [Live Portfolio Link](https://https://portfolioishu.netlify.app/)

**Kuverdeep Pundir's Portfolio** is a sleek, modern, and fully responsive digital space showcasing my expertise in **Full-Stack Development**. Built with high-performance Vanilla JS and custom CSS, this site highlights my technical skills, academic background, and production-grade projects.

---

## 🚀 Key Features

* **Dynamic Hero Section**: Features a floating avatar UI with custom CSS animations and a "Neon-Red" accent theme.
* **Fully Responsive Engine**: Engineered with custom Media Queries for a pixel-perfect experience on Mobile, Tablet, and Desktop.
* **Interactive About Workspace**: A tabbed interface system allowing users to switch between Skills, Education, and Achievements instantly.
* **Functional Contact Form**: Integrated with a custom **Google Sheets Backend** for real-time message delivery without an external server.
* **Optimized Work Showcase**: Hover-animated project cards with external link integration for production apps.

## 💻 Tech Stack & Knowledge Used

* **Frontend Architecture**: HTML5 & CSS3 (Flexbox/Grid/Animations)
* **Logic Layer**: Vanilla JavaScript (DOM Manipulation & ES6+)
* **Form Handling**: Google Apps Script (REST API)
* **Icons**: Font Awesome 6.x
* **Deployment**: Netlify

## 🛡️ Form Integration & Scripting

To ensure the contact form works securely and efficiently:
* **Google Apps Script**: Processes form data and appends it to a secure Google Sheet.
* **Asynchronous Fetch**: Uses the JavaScript `Fetch API` to submit data without refreshing the page, providing a smooth UX.

## 🛠️ Installation & Local Setup

1.  **Clone the repo:**
    ```bash
    git clone (https://github.com/ishurana001/personal-portfolio.git)
    ```

2.  **Open the project:**
    Simply open the `index.html` file in your preferred browser or use **Live Server** in VS Code.

3.  **Configure Contact Form (Optional):**
    To connect your own Google Sheet:
    * Create a new Google Sheet.
    * Use `Extensions > Apps Script` to add the handler code.
    * Deploy as a Web App and replace the `scriptURL` in `index.html`.
